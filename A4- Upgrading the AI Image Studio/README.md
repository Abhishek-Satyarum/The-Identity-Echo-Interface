# AI Image Studio

## Features

- Generate AI images from text prompts
- Multiple artistic styles
- Adjustable image width and height
- Magic Enhance mode for detailed image generation
- Surprise Me feature with random creative prompts
- Download generated images as PNG
- Powered by Pollinations AI
- Clean and interactive Streamlit interface

---

## Available Art Styles

- Photorealistic
- Water World
- Cartoon
- Sketch
- 3D Render

---

## Tech Stack

- Python
- Streamlit
- Requests
- Pollinations AI API

## How to Use

1. Select an art style from the sidebar.
2. Adjust the image width and height.
3. (Optional) Enable **✨ Magic Enhance** for more detailed results.
4. Enter your image description.
5. Click **Generate Image**.
6. Or click **🎲 Surprise Me!** to generate a random creative image.
7. Download your generated image as a PNG file.
